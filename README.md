## 什么是 AX Video?


> **AX Video 爱写视频**，是一个桌面软件，自动将 Markdown 文档转成一段视频的短视频编辑器，支持操作界面
文档里面的视频、音频、图片网址，都会抓取后插入视频，还可以根据文字生成人工语音的旁白朗读。
